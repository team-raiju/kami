Sobre nosso labirinto de testes (tamanho 7x4 células), compramos uma chapa de MDF preta de 1300x750mm de 6mm para a base do labirinto e 45 chapas pequenas de 168x50mm brancas de 6mm para as paredes.

A medida oficial de um labirinto é com madeiras de 12mm, mas para testes 6mm é suficiente, o único ponto negativo é que os postes tem que ser ajustados em tamanho.

Do jeito que fizemos certas partes dos postes ficam com protuberancias para dentro do labirinto, mas não chegaram a atrapalhar os testes.

Mandamos as madeiras virem ja cortadas por uma loja chamada "[LIVINGSTONE FISCHER](https://lista.mercadolivre.com.br/_CustId_1106083327?item_id=MLB2611640066&category_id=MLB251200&seller_id=1106083327&client=recoview-selleritems&recos_listing=true#origin=upp&component=sellerData&typeSeller=classic)" do mercado livre.
