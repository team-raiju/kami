Para desenvolver o algoritmo de flood fill e otimizar para nosso robô, utlizamos o simulador [MMS](https://github.com/mackorone/mms). Ele permite rodar códigos em diversas linguagens que vão conversar por uma API com o simulador.

Usar o simulador ajuda bastante para isolar problemas de software dos problemas de hardware.

Para o nosso programa que roda com base no MMS, temos esse repositório: [Fujin MMS](https://github.com/team-raiju/fujin-mms)
